The Roofing Contractor's Journey: Current State

Sarah's Story: A Typical Roofing Contractor

Sarah runs a 5-person roofing company. For each new project, she must:

  1. Check Zoning (30 min): Navigate city website to find property's zoning classification
  2. Research Requirements (60 min): Read through code to understand height limits, materials
  3. Historic Check (45 min): Determine if property is in historic/conservation district
  4. Document Prep (45 min): Gather owner info, create work description, find old permits
  5. Submit & Wait (1 week): Hope nothing was missed that causes rejection

Total: 3 hours per permit × 50 permits/year = 150 hours = $7,500 in overhead

Roofing-Specific Challenges & Solutions

Challenge: Storm Damage Surge

After hail storms, permit applications can spike 300% in days

Solution: Bulk Query Mode

  • Upload multiple addresses via CSV
  • Batch process requirements
  • Generate permit packages for each
  • Flag any special requirements

Challenge: Historic Districts

11 historic districts + 45 conservation districts = complex rules

Solution: Visual Guidance

  • Photo examples of approved materials
  • Color palettes by district
  • Direct link to preservation staff
  • Previous approvals database
