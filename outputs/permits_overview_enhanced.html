Contractor Pain Points Analysis

1. Information Fragmentation

Contractors must navigate 14 different city websites and documents to gather complete permit requirements. The average project requires consulting:

  • Zoning code (476 pages)
  • Building code references (multiple volumes)
  • Overlay district requirements
  • Historic preservation guidelines
  • Environmental regulations

2. Interpretation Uncertainty

37% of contractors report "lack of clarity" as their primary challenge. Common issues include:

  • Conflicting requirements between departments
  • Vague language requiring staff interpretation
  • Updates not clearly communicated
  • Project-specific nuances not documented

3. Process Inefficiencies

The current permitting workflow creates multiple friction points:

  • Pre-submission: 8-12 hours of research per project
  • Submission: 2-3 hours preparing documentation
  • Corrections: 65% require resubmission (4-6 hours)
  • Follow-up: Average 3.2 calls to city staff
